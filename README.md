# 🎮 Game Title: Push to Solve

## 📌 Genre
Third-Person Physics Puzzle Game

## 💻 Platform
PC (Keyboard & Mouse)

## ⚙️ Engine
Unreal Engine

---

## 🧠 Game Description
This is a third-person physics-based puzzle game focused on object interaction, weight-based mechanics, and environmental challenges.

Players must observe the environment, identify different types of boxes, and push the correct objects onto specific pressure plates to complete each level.

The game emphasizes problem-solving, planning, and decision-making.

---

## 🎯 Core Gameplay Features
- Third-person character movement and camera control  
- Physics-based box pushing system  
- Weight-based puzzle mechanics  
- Multiple pressure plates with different requirements  
- Environmental obstacles affecting movement  
- Visual and audio feedback  

---

## 🎮 Controls
| Key | Action |
|-----|--------|
| W / A / S / D | Move |
| Mouse | Rotate Camera |
| Space | Jump |
| Shift | Run / Increase push force |
| E | Push objects |

---

## 🧩 Puzzle Mechanics

### Box Types
- **Heavy Box** → activates **Red Plate**  
- **Light Box** → activates **Purple Plate**

### Rule
All required pressure plates must be activated correctly to complete a level.

---

## 🌍 Environmental Mechanics
- **Spider Web Areas**
  - Reduce movement speed  
  - Increase puzzle difficulty  

---

## 🔊 Audio Design
- Footstep sound  
- Jump sound  
- Box pushing sound  
- Level completion sound  

---

## 🗺️ Level Design & Progression

### Level 1
- Basic movement & pushing

### Level 2
- Introduces weight-based decisions

### Level 3
- Inclined platform challenge

### Final Level
- Combines all mechanics + obstacles

---

## 🏆 Win Condition
- All pressure plates activated correctly  

➡️ Next level loads automatically
